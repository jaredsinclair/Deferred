Task

public final class Task<SuccessValue>: NSObject

A wrapper over any task.

Forwards operations to an arbitrary underlying future having the same result type, optionally combined with some cancellation.

  • cancel()

    Attempt to cancel the underlying operation.

    An implementation should be a best effort. There are several situations in which cancellation may not happen:

  • The operation has already completed.
  • The operation has entered an uncancelable state.

  • The underlying task is not cancellable.
